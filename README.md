# TA Programme Turnup Portal - NUnit Automation

## 📌 Project Overview
This project contains automated regression tests for the Turnup Portal using Selenium WebDriver with C# and NUnit.

## 🛠 Tech Stack
- C#
- Selenium WebDriver
- NUnit
- Visual Studio

## 🧱 Framework Design
- Page Object Model (POM)
- Reusable test methods
- Centralised driver management

## ✅ Features
- Create Time & Material record
- Edit Time & Material record
- Delete Time & Material record
- Assertions for validation
- Dynamic XPath handling
- Wait strategies (Explicit Wait)

## 🎯 Purpose
This project is part of my Test Analyst learning journey, focusing on building real-world automation testing skills.
